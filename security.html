Security SLip

Security SLip is inspire by game called Papers, Please. You work as a security decide if poeple who want to enter the building can go inside if their entry card is valid.

Security SLip is a project created using Unity Engine and using C# as programming language.

As a team project, I was responsible of designing the user interface and making the people database.

I design the visual elements which player can interect while player playing the game.

I create a database where random name will genereate on the database, so different name will always appear in the database.
