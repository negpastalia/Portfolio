Mate Matirun

A math learning based rhythm game, where you do the rhythm while doing some math question at the same time until the end of the song.

Mate Matirun is a whythm game created using Unity Engine and using C# as programming language

As a team project, I was responsible of animating characters, making difficulty, and leaderboard highscore.
